# Сим‑нарратив: Репозиторий учебных и научных работ

Сборник академических работ по **методу сим‑нарратив**: курсовые, дипломные, эссе, шаблоны анализов, исследовательские материалы.

## Содержание

- `templates/` — шаблоны для курсовых, дипломных и анализа слоёв.
- `examples/` — примеры студенческих работ (курсовые, дипломы) о сим‑нарративе как литературном методе и жанре.
- `sources/` — библиография и списки источников.

## Назначение

Репозиторий предназначен для:
- Студентов и исследователей, изучающих новые литературные методы.
- Преподавателей, ищущих учебные материалы по сим‑нарративу.
- Писателей, которым важно академическое обоснование метода.
