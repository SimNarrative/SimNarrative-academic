# 📚 SimNarrative / academic

Сборник учебных и научных работ по методу сим‑нарратив: курсовые, дипломные, эссе, шаблоны анализов.  
Материалы для студентов, преподавателей и исследователей нового литературного инструментария.

## Связанные проекты

- [Официальный сайт сим‑нарратива](https://simnarrative.github.io/) – главная страница экосистемы.
- [Философский манифест](https://simnarrative.github.io/philosophy-manifesto.html) – «Чат длиною в жизнь».
- [Литературный манифест](https://simnarrative.github.io/literary-manifesto.html) – три слоя, ветви, этика.
- [AI‑манифест](https://simnarrative.github.io/ai-manifesto.html) – многослойные тексты для обучения ИИ.
- [Сборник рассказов](https://simnarrative.github.io/stories-collection.html) – 12 живых доказательств метода.

## Содержание репозитория

- `/templates` – шаблоны курсовых и дипломных работ.
- `/examples` – готовые примеры анализа рассказов через метод.
- `/sources` – библиография и дополнительные материалы.
